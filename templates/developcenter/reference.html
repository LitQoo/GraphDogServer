Setup

1. Download Graphdog-client source at github

    http://github.com/LitQoo/GraphDog

2. setup curl lib

How to start

graphdog -> setup ( string  appID, string  secretKey, string  deviceId );

Prameter Descript Type
appID app ID String
secretKey secret key String, length=8
deviceId device ID, like udid String
Example!
graphdog -> setup( "myapp" , "GDKEY123"  , myudid() );

How to use

graphdog -> command( string  action,  const   JsonBox :: Object *  const  param, CCObject *target, GDSelType  selector);

Prameter Descript Type
action action String
param secret key JsonBox::Object or Null
target callback target CCObject or Null
select callback selector GDSelType or Null
Example!
graphdog -> command( "timestamp" , null  , this , gd_selector(class::callbackfunction) );
